Начни Писать Код На JavaScript


Язык JavaScript предназначен для выполнения в браузере наряду с HTML и CSS. Но, если эти языки предназначены для верстки структуры сайта, то JavaScript позволяет 'оживлять' web-страницы - делать их реагирующими на действия пользователя или демонстрировать некоторую динамичность (к примеру, смена картинок в блоке или красивые плавно выпадающие менюшки).

Ниже представлены примеры того, что можно сделать на JavaScript.

Строка, которая написана в форме заключается в переменную методом "document.getElementById("").value". Затем эта строка проверяется на пустоту отдельно написаной функцией, в которой строка сравнивается с null, "" и undefined. Когда строка проходит проверку, она записывается куда нужно, спомощью метода "document.getElementById('my_name').innerHTML".

Строка, которая была написана последней, хранится в браузере с помощью localStorage. После того как строка проходит проверку её значение заноситься в хранилище функцией "localStorage.setItem("ключ", строка);". Во-время загрузки стайта запускается функция, которая берет значение из хранилища и записывает её в правый верхний угол сайта (" document.getElementById().innerHTML = localStorage.getItem(ключ) ")

Берет значения из форм так же как и в первом задании. Затем идет проверка на число. В функции проверку=и строки преобразуются в числа (x = Number(x)) затем идет проверка на NaN (если в строке есть хотя бы один символ, искоючение пара символов, то при преобразовании в тип Number вернет Nan). Проверка делается функцией isNan(). После проверки выводит на экран площадь треугольника (0,5 * высоту * основание).

При нажатии на кнопку добавить, берется значение из инпута и проверяется. Если число прошло проверку, то оно добавляется в массив функцие push(). Если добавляется шестое число в массив, то первое число удаляется функцией shift().

Таймер

Нажав на кнопку запуска, значения из инпутов кладутся в переменные и эти переменные проверяются. После проверки запускается функция обратного отсчета, которая будет повторяться определённое время. Делается это с помощью метода setInterval( функция, промежуток времени ).

При нажатии на паузу запускается функция clearInterval(), которая останавливает повторяющуюся функцию в setInterval(). А кнопка переименовывается в кнопку "Возобновить". При нажатии на неё обратный отсчет возобновляется, а кнопка переименовывается обратно. Чтобы это можно было контролировать была создана булевая переменная isPause.

При нажатии на кнопку "Выключить", обратный отсчет останавливается, а все переменные обнуляются.

В функции проверки вначале запускается цикл, с помощью которого мы можем пройти по каждому вопросу отдельно. Зате создаем переменную которая принимает массив вариантов ответа на этот вопрос, для этого используется метод getElementsByClassName("повторяющаяся часть"+номер вопроса). Дальше идёт обход этого массива и и идет проверка наших ответов. Свойство checked позволяет на найти вариант ответа которые мы выбрали. Затем проверяем чему равен атрибут value у этого переключателя (их значения записаны в html-коде). Если ответ правильный, то переменная отвечающая за кол-во правильных ответов увеличивается. В конце количество правильных ответов выводиться на экран.

Покраску правильных и неправильных ответов выполнаяет отдельная функция. Работает она также как и функция описанная выше. Только теперь мы создаём ещё массив тектовой части вариантов ответа. Все правильные ответы красятся в зелёный, а наши неправильные в красный. Меняем цвета с помощью изменения свойства style у нашего объекта.

При нажатии на кнопку, меняется высота спратанной плашки (element.style = "CSS стили"). Плавность появления обеспечивает CSS-свойство transition, которая отвечает за эффект перехода. В нашем случае было указано только время за которое объект должен перейти из одного состояние в другое.
